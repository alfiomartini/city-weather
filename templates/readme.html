This web application provides current weather information and a seven day forecast for around 15000 cities in the world. It is implemented with HTML, CSS, JavaScript, Bootstrap, Python, Flask and the SQLite database. As the user types in the search input text area, a list of cities is displayed. By clicking on any list item, the user is presented with the weather information for the given city. The database of cities is built on top of free .csv files provided by Simple Maps. Weather data and parameters are collected using the free API published by openweathermap.org. You can see the app live here.
